For OS X, libusb is required (http://www.libusb.org/).

Easiest way to obtain libusb on OS X is macports OR homebrew.

-Macports Install:
--open terminal
--update macports. (in terminal, enter below):
	sudo port selfupdate
--obtain libusb via macports. (in terminal, enter below:):
	sudo port install libusb

Homebrew Install:
--open terminal
--update homebrew. (in terminal, enter below):
	sudo brew update
--obtain libusb via homebrew. (in terminal, enter below:):
	sudo brew install libusb
